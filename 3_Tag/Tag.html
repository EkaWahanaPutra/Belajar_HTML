Pada materi kali ini kita akan membahas tentang tag pada html

tag yang pertama : adalah "!doctype html" ==>> tag ini menjelaskan bahwa kita sedang menggunakan html terbaru

tag yang kedua : adalah "head" didalam tag head, kita bisa menggunakan tag "title" untuk membuat judul pada tab halaman. Kemudian tag "style" untuk menyimpan CSS, tag "script" untuk mennyimpan javascript, dan tag "meta" untuk mediskripsikan web kita

tag yang ketiga adalah tag "body". Tag body merupakan isi dari web kita. Pada tag ini, kita bisa mengisi berbagai macam tag (akan dibahas pada latihan latihan berikutnya). Salah satu tag pada body adalah tag "p". Yang berfungsi untuk membuat paragraf
